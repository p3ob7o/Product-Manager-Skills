## How To Add A New Announcement

1. Create `docs/announcements/YYYY-MM-DD-<channel>-<topic>.md`
2. Include:
   - post metadata (title, subtitle, opening)
   - short promotional post
   - long-form article/draft copy (if relevant)
3. Add the file to `Latest` and `Timeline` in this index.
4. Add or update the top `📣 Updates & Announcements` section in `README.md`.
